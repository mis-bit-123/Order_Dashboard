# Order Dashboard

A Streamlit dashboard for visualizing order data from Google Sheets.

## Setup Instructions

1. Add your Google Service Account JSON to HuggingFace Secrets as `GOOGLE_SERVICE_ACCOUNT`
2. Deploy and enjoy!

## Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
